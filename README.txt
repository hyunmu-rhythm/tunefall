================================================================
                    TUNEFALL - 리듬게임
================================================================

[ 실행 방법 ]
TUNEFALL .exe 를 실행해주세요.

[ 최초 실행 안내 ]
처음 실행 시 아래 파일이 자동으로 다운로드됩니다.
인터넷 연결이 필요하며 시간이 걸릴 수 있습니다.

  - ffmpeg (영상 변환 도구, 약 100MB)
  - Spleeter 모델 (채보 생성 AI, 약 500MB)

다운로드가 완료될 때까지 기다려주세요.

[ 곡 추가 방법 ]
1. 곡 선택 화면에서 유튜브 URL 을 입력해주세요.
2. ADD 버튼을 클릭하면 자동으로 다운로드됩니다.
3. 채보 생성 완료 후 곡 목록에 추가됩니다.
4. 영상 변환은 백그라운드에서 진행되며
   완료 후 영상 ON 버튼이 활성화됩니다.

[ 채보 공유 방법 ]
  같은 유튜브 URL 로 ADD 하면 동일한 채보가 생성됩니다.
  친구에게 유튜브 URL 을 공유하면 같은 채보로 플레이할 수 있어요!

  직접 채보 파일을 공유하고 싶다면:
  1. assets/songs/[곡제목]/ 폴더를 압축합니다.
  2. 받는 사람은 압축을 풀어
     TUNEFALL/assets/songs/ 안에 넣어주세요.
  3. 게임을 실행하면 곡 목록에 자동으로 추가됩니다.

  * 단, 받는 사람도 같은 버전의 TUNEFALL 을 사용해야 합니다.

[ 기본 조작 방법 ]
  1번 레인 : D
  2번 레인 : F
  3번 레인 : J
  4번 레인 : K

  * 설정 화면에서 키를 변경할 수 있습니다.

[ 난이도 ]
  EASY   : 입문자용
  NORMAL : 일반
  HARD   : 고수용

[ 기록 데이터 저장 위치 ]
  게임 기록과 설정은 아래 경로에 저장됩니다.

  C:\Users\[사용자이름]\AppData\Roaming\Godot\app_userdata\TUNEFALL\

  records.json  → 곡별 최고 점수, 최고 콤보, 플레이 횟수
  settings.json → 키 설정, 볼륨, 노트 속도

  * 데이터를 초기화하려면 위 파일들을 삭제해주세요.

[ 업데이트 방법 ]
  1. 새 버전의 TUNEFALL 폴더를 다운로드합니다.
  2. 기존 폴더에서 아래 항목을 새 폴더로 복사합니다.

     assets/songs/          → 기존에 추가한 곡들

  3. 새 버전의 TUNEFALL.exe 를 실행합니다.

  * 기록은 별도 경로에 저장되므로 업데이트해도 유지됩니다.

[문의사항]
 문의사항은 메인화면의 디스코드를 이용해주세요.
