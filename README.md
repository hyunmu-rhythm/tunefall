# TUNEFALL 🎵

유튜브 URL 로 곡을 추가하고 플레이하는 리듬게임입니다.

## 다운로드

[TUNEFALL_Setup.exe 다운로드](https://github.com/hyunmu-rhythm/tunefall/releases/latest/download/TUNEFALL_Setup.exe)

## 설치 방법

`TUNEFALL_Setup.exe` 를 실행하면 자동으로 설치됩니다.

## 최초 실행 안내

처음 실행 시 아래 파일이 자동으로 다운로드됩니다.
인터넷 연결이 필요하며 시간이 걸릴 수 있습니다.

- ffmpeg (영상 변환 도구, 약 100MB)
- Spleeter 모델 (채보 생성 AI, 약 500MB)

## 곡 추가 방법

1. 곡 선택 화면에서 유튜브 URL 을 입력해주세요.
2. ADD 버튼을 클릭하면 자동으로 다운로드됩니다.
3. 채보 생성 완료 후 곡 목록에 추가됩니다.
4. 영상 변환은 백그라운드에서 진행되며 완료 후 영상 ON 버튼이 활성화됩니다.

## 채보 공유 방법

같은 유튜브 URL 로 ADD 하면 동일한 채보가 생성됩니다.

직접 채보 파일을 공유하고 싶다면:
1. 곡 선택 화면에서 공유할 곡을 선택 후 공유 버튼을 클릭합니다.
2. zip 파일이 생성되면 친구에게 전달해주세요.
3. 받는 사람은 zip 을 풀어 아래 경로에 넣어주세요.
   `C:\Users\[사용자이름]\AppData\Roaming\Godot\app_userdata\TUNEFALL\songs\`
4. 게임을 실행하면 곡 목록에 자동으로 추가됩니다.

## 기본 조작 방법

| 키 | 기능 |
|---|---|
| D | 1번 레인 |
| F | 2번 레인 |
| J | 3번 레인 |
| K | 4번 레인 |
| ESC | 일시정지 |

설정 화면에서 키를 변경할 수 있습니다.

## 난이도

- NORMAL : 일반
